Some time off over Easter meant I was able to spend some time getting started with the [Bantam NextDraw 8511](https://bantamtools.com). This week was mostly about setting up the plotter in a slightly more accessible location and getting started with setting up inkscape and [place to record progress](https://www.iot.io/plotter/). Hopefully to make it easier to continue than my [false start last time](https://www.iot.io/blog/2025/01/18/plotter.html).

<!--more-->

So far I have been getting the workflow sorted. I created the website at [https://www.iot.io/plotter/](https://www.iot.io/plotter/) since I was keen to store how I was making different plots and wanted to keep images of things created. I am curious about the algorithms behind the algorithmic art.
